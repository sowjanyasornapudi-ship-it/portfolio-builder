# **Portfolio Website Prompt for Lovable**

Create a premium, modern, responsive personal portfolio website for a Data Engineer / Cloud Data Engineer professional. 

## **Objective**

The website should act as a personal brand website showcasing:

* Professional profile  
* Educational background  
* Technical skills  
* Work experience  
* Projects  
* GitHub repositories  
* Certifications  
* Achievements  
* Resume  
* Contact information  
* Social media profiles  
* Professional journey

The design should feel like a combination of modern SaaS websites, developer portfolios, and consulting websites.

---

## **Design Style**

* Modern and premium  
* Clean UI with smooth animations  
* Dark mode by default  
* Glassmorphism effects  
* Professional gradients  
* Mobile-first responsive design  
* Fast loading  
* SEO optimized  
* Accessibility compliant

Inspirations:

* Apple-style minimalism  
* Stripe-style sections  
* Linear-style animations  
* Vercel-style clean layouts  
* Modern developer portfolio aesthetics
